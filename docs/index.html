About echoevm

EchoEVM is a from-scratch implementation of the Ethereum Virtual Machine written in Go. It's designed to be educational, minimal, and focused on understanding how EVM bytecode execution works.

The project started as a learning exercise to understand the internals of the EVM, from basic opcode execution to complex smart contract interactions.

Educational Minimal From Scratch Open Source 
$ echoevm call \
  -a ./artifacts/Add.json \
  -f "add(uint256,uint256)" \

Quick Start

Get up and running in minutes

01 Installation 

# Install from source
go install github.com/smallyunet/echoevm/cmd/echoevm@latest

# Or clone and build
git clone https://github.com/smallyunet/echoevm.git
cd echoevm
make build

02 Run Bytecode 

# Simple arithmetic: PUSH1 1 PUSH1 2 ADD
echoevm run 6001600201

# With debug trace
echoevm run --debug 6001600201

03 Deploy & Call 

# Deploy a contract
echoevm deploy -a ./artifacts/Add.json --print

# Call a function
echoevm call -a ./artifacts/Add.json \
  -f add(uint256,uint256) -A 2,40

04 Trace Execution 

# Generate execution trace
echoevm trace -a ./artifacts/Add.json \
  -f add(uint256,uint256) -A 1,2 \
  --limit 20 | jq .
